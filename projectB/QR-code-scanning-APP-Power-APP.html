QR code scanning APP(Power APP)

For easier to manage inventory in warehouse, developed a mobile app (based on MS Power Apps) that allows warehouse users
to scan QR code from goods and upload those records to Google Sheet(Excel) as for inventory recording.

If the uploaded record can be found in a large data sets from the seperated page in the same Google Sheet file.
As for look-up purpose, the associated info(e.g Customer No. , Name , Address , etc) should also be returned.

(Click the pictures to enlarge)

Mobile App

Once scanned QR code,
the procedure will be:

1. The serverless App Script(JavaScript) will receive a HTTP request, which is attached the info of scanned QR code.

2. The latest uploaded info of QR code should append on pages of Google Sheet.

3. Based on the uploaded info, the serverless App Script(JavaScript) will also look-up the associated info(e.g Customer No. , Name , Address , etc) from the seperated page. If found, it will retrun the look-up result back to mobile app.

Google Sheet

All the uploaded QR code can be found in Google Sheet. Users can collect these data and place to other file if need.

Users can also update the data sets with associated info(e.g Customer No. , Name , Address , etc) by replacing new page with proper format in Google Sheet.

It's much convenient for users if only know Excel but don't know how to manage database(SQL/NoSQL).
